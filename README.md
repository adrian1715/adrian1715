## 🚀 Featured Projects

### 🎯 [Producer - Focus & Productivity Chrome Extension](https://github.com/adrian1715/producer-extension)

> **Founder & Solo Developer - Chrome SaaS Extension**

A productivity tool that blocks distracting websites and URLs with granular control, built on a freemium model with a paid tier in progress. Actively developed with monthly build reports since December 2025.

**Tech Stack:** TypeScript • Chrome Extension API • Manifest V3

**Key Features:**

- 🗂️ **Modes** - named, switchable rule sets for different contexts (Deep Work, Studying, etc.)
- ⏱️ Timed **focus sessions** with history, stats, and averages
- 🛡️ Flexible rules: block domains, specific URLs, query-parameter allow rules, permanent (always-on) rules, block-all wildcard
- 🎨 Fully customizable block page - themes, colors, custom copy, motivational quotes, background image
- 🌓 Per-mode features like **Grayscale** to reduce a page's pull during focus
- 🔁 Import/export rule sets between machines
- 🔒 Privacy-first - all data stored locally, no accounts, no analytics

**Chrome Web Store:** [Producer - Focus Extension](https://chromewebstore.google.com/detail/producer-free-focus-exten/ljihbhechnhegaokmpkhmeonhmlmiijk)
